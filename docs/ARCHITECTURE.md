# CompText DSL Architecture

## Components

### Parser
- Tokenization
- Syntax analysis
- AST generation

### Compiler
- AST optimization
- Code generation
- Output formatting

### Runtime
- Command resolution
- Module loading
- Context injection

## Integration

CompText DSL integrates with:
- [MCP Server](https://github.com/ProfRandom92/comptext-mcp-server)
- Notion Codex
- Various LLM clients
